SmartNougat OMML 변환 예시

OMML 변환 기능 설명

LaTeX 수식을 Microsoft Word의 수식 편집기에서 사용할 수 있는 형식으로 변환합니다.

사용법: OMML 형식을 복사한 후 Word에서 Alt+= 를 눌러 수식 편집기를 열고 붙여넣기하세요.

#1 분수 예시

원본 LaTeX

\frac{a+b}{c-d}

Word 수식 편집기용 형식 / OMML Format for Word

(a+b)/(c-d)
#2 제곱근 예시

원본 LaTeX

\sqrt{x^2 + y^2}

Word 수식 편집기용 형식 / OMML Format for Word

√(x² + y²)
#3 그리스 문자와 첨자

원본 LaTeX

\alpha_1 + \beta^2 = \gamma_{n+1}

Word 수식 편집기용 형식 / OMML Format for Word

α₁ + β² = γ_{n+1}
#4 적분 예시

원본 LaTeX

\int_{0}^{\infty} e^{-x^2} dx = \frac{\sqrt{\pi}}{2}

Word 수식 편집기용 형식 / OMML Format for Word

∫₀^∞ e^(-x²) dx = √π/2
#5 행렬 예시

원본 LaTeX

\begin{pmatrix} a & b \\ c & d \end{pmatrix}

Word 수식 편집기용 형식 / OMML Format for Word

( a b c d )
